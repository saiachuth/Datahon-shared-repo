# Seattle Edge Accessibility Dashboard

Government dashboard for monitoring sidewalk and accessibility issues across Seattle.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Features

- **Home**: Interactive map of Seattle edges color-coded by barrier count
- **Issue Metrics**: Filter by issue type (CurbRamp, NoSidewalk, etc.) and view severity distribution
